Waykup!

I wanted to develop a javascript app for mobile that performed as a native app focusing on every detail for the user experience to feel native like, every touch input, gesture, animation, speed and the UI. The technology stack used for development: Apache Cordova to wrap the webapp to be installed as native, grunt.js for compiling, require.js for modularity of the js code, hammer.js for touch gestures and sass as the css pre-precessor. The app consists on a geolocation alarm. It is currently available in the iOS appstore. Download now!

BingoCrack

HTML5 Video Game

It's a real-time bingo game with sockets.

Developed the whole UI of the game, including all animations. First made all of them with the greenSock JS animation library, but they weren't performing as fluid as desired, so I migrated them to CSS3 using hardware accelerated animations. Compared performance of each one of them cross-browser using developer tools and fps meters. In conclusion made the perfect blend between greenSock and CSS3 animations.
